Feedback badge demo

Features

  • Custom styling, positioning, images and messages
  • Multiple open directions
  • Handles resizing comment field
  • Character limits on form inputs
  • Endpoint posting via fetch
  • Element attributes to set options or run via JS object
  • Options can be used globally and overriden per element
  • Shares email field between all instances
  • Sets focus on comment field on opening
  • Accessibility highlights on focus (tabindex)

Usage

The badges can be added as elements you need then run the plugin with the class or id. <div class="my-feedback-badge"></div> window.badgerFeedback.addAll('.my-feedback-badge'); You must implement a fetch polyfill to use the endpoint option.
